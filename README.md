# Hideout Hunter

A Scalpel plugin to help you find rare hideouts from maps in PoE 1

## Usage

Pick a map that has a hideout you want, open the overlay, and enter the map. When the overlay shows `HIDEOUT: Yes` that means a hideout spawned in the map, now all you have to do is find it. If no hideout spawned then simply exit the map and open a new one.

This simplifies the proccess of finding hideouts in maps, as you no longer have to clear the whole map manually.
